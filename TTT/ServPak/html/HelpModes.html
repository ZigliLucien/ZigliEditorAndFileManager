Modes

Here we present the basic built-in Modes. In the Utils Menu or by Ctrl-Shift-right click on a buffer window, you will see a list of the modes:
Jxml, TeX, Perl, Clsv, Xml, LaTeX, Jv, Html, Text, Java, Ant, and Lang.

These are prescribed in the Modes.properties file. First are the compile commands (on F4) for compiling or running the file.

Each mode has five associated auxiliary files. Here [Mode] denotes the name of the mode.
Strings.data.[Mode]
Quickwords.data.[Mode]
[Mode].data
[Mode].help
[Mode].keywords
