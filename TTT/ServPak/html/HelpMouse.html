Mouse Bindings for Zigli's Editor

Double Click of either left or right button will refresh the buffer. Useful if the keyboard locks up.
All the rest are single click.

 

Left Mouse

 

Modifier or Context    Command
In background window    Expands a single window. Tiles multiple windows.
In buffer    Sets current file name in status line
Shift    Highlight from current cursor position
Ctrl    Cycle buffers
Alt    Paste

 

Right Mouse

 

Modifier or Context    Command
In background window    New Buffer (click on `Cancel' to start with a default file)
On a menu item or clickpad item    Brings up Help for that item.
Buffer background    Brings up UniPop Menu: first line shows font information, any Unicode menus chosen from the UniBlox master menu will appear here along with alphabet, clickpad, and quickwords menus. Includes Modes Menu for setting the mode.
Shift, on a word or highlighted text    Highlights. Brings up the Dictionary/Thesaurus menu for that selection.
Ctrl    Cut highlighted text
Alt-Shift    Brings up Dictionary-In dialog box for entering Dictionary/Thesaurus information for word or selection.

 

Middle Mouse

 

Modifier or Context    Command
Buffer    Paste
